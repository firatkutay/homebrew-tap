# homebrew-tap

Homebrew tap for [`comrade`](https://github.com/firatkutay/cli-comrade) —
a cross-platform AI CLI companion for the terminal (TR/EN).

Published automatically by `cli-comrade`'s `goreleaser` release pipeline
(`homebrew_casks` block in `.goreleaser.yaml`). Do not edit `Casks/*.rb` by
hand — every release overwrites it.

## Install

```sh
brew tap firatkutay/tap
brew install comrade
```
